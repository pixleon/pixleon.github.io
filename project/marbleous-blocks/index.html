Marbleous Blocks

Sep 7, 2019

Marbleous Blocks is a 3D-puzzle game where you need to carefully construct paths using blocks in order to free your marbleous marble friends. The game is currently in development and scheduled for release on PC during 2020.

Marbleous Blocks has taken inspiration from a number of games including Patchwork, The Witness and Super Metroid.

If you want to try the beta version please contact Pixleon.
